Mine using your web browser with our web proxy cpu miner

Use our web miner at https://proxy.mpool.stream . Address, worker, threads, and throttle can be set by url parameters and it mines in your web browser.

https://proxy.mpool.stream/?throttle=[thottle]&threads=[threads]&worker=[worker]&address=[address]
- throttle (0.01 - 0.99) - the lower the value the more cpu it uses (0.01 would use all the cpu. 0.25 would use about 75% cpu. 0.5 would used about 50% cpu)
- threads - number of cores you want to use (0 is unlimited)
- worker - the worker name your miner will be given in the pool
- address - the monero address you want to use in the pool
